程序的构成
----------

由一堆 **指令** 构成：

  - 输入
    
    `从键盘设备、文件、或其他设备获取数据`
    
  - 输出
  
    `输出到屏幕、写入一个文件、发送到其他设备等`
    
  - 基本运算
  
    `加减乘除、数据存取`
  
  - 测试和分支
  
    `测试条件，根据不同测试结果，执行不同后续指令`
  
  - 循环
  
    `重复执行操作`
    
就是以上这么几条简单的指令类型，构成了复杂的程序。
