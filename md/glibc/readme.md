C标准库 和 glibc
============================

C标准
----------

由两部分组成：
  
  - **C语法**
    
    - 定义了语法的结构标准，程序员秒懂
  
  - **C标准库**
    
    - 定义了一组标准*头文件*
    
      进而定义了*每个*头文件中相关的
      
      - 函数
      - 变量
      - 类型声明
      - 宏定义
      
      
符合C标准的平台
----------
  
  - 实现C编译器
  
  - 实现C标准库
  
  
例外，不符合C标准的平台
----------
  
  - 单片机
    
    `许多单片机只有C编译器，而没有完整的C标准库`
  
  - 其它
  
glibc
----------
glibc 是Linux平台上最广泛使用的C函数库。而且它囊括了**C标准库的实现**
