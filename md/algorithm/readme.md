递归
----------

自我调用，数学上确实有很多概念是用到自己调用自己的定义，这就是递归。

  - 关键：需要定义一个**基础条件**（Base Case）

    即*终止条件*；

    否则递归将会没完没了的死循环；

    ```c
    0! = 1
    ```

    并且

    ```c
    1! = 1        // （1 * 0!） = 1 * 1
    ```

  - 典型例子，无疑是阶乘

    方法

      1.先把*基础条件*写上，定好框架

        `````c
        int factorial(int n)
        {
          // Base Case.
          if (n == 0)
          {
            return 1;
          }
        }
        `````
