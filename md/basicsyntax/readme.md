循环语法
----------

各种循环语法之间，是完全可以相互互换的！

- for循环

  ```c
  语法是

  语句 ---> for(控制表达式1，控制表达式2，控制表达式3) 语句;
  ```

  可以替换为 while 语法：

  ```c
  控制表达式1;

  while(控制表达式2)
  {
    语句;
    控制表达式3;     // 如 i++
  }
  ```

- while循环

- do/while循环
